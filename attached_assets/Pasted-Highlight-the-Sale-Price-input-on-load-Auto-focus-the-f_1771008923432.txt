Step 2: Add this CSS to your <style> tag (or Tailwind config / custom CSS file)
CSS/* Pulse once for 4 seconds on load – gentle breathing effect */
.animate-pulse-once {
  animation: pulseOnce 4s ease-in-out forwards;
}

@keyframes pulseOnce {
  0%   { box-shadow: 0 0 0 0 rgba(34, 197, 94, 0.4); }
  20%  { box-shadow: 0 0 0 12px rgba(34, 197, 94, 0.25); }
  40%  { box-shadow: 0 0 0 8px rgba(34, 197, 94, 0.15);  }
  60%  { box-shadow: 0 0 0 12px rgba(34, 197, 94, 0.25); }
  80%  { box-shadow: 0 0 0 6px rgba(34, 197, 94, 0.1);  }
  100% { box-shadow: 0 0 0 0 rgba(34, 197, 94, 0);       }
}
(Uses your green theme – #22c55e / green-500. Adjust colors/duration if desired.)
Step 3: Add this small JavaScript snippet
Paste this inside your existing <script> tag (preferably near the bottom, after your updateAll() logic):
JavaScript// Highlight Sale Price on load
document.addEventListener('DOMContentLoaded', () => {
  const saleInput = document.getElementById('sale-price');
  const callout   = document.getElementById('sale-price-callout');

  if (saleInput) {
    // 1. Auto-focus
    saleInput.focus();

    // 2. Select text so user can type over it immediately
    saleInput.select();

    // 3. Show callout after tiny delay (feels more natural)
    setTimeout(() => {
      if (callout) {
        callout.classList.remove('opacity-0', 'scale-95');
        callout.classList.add('opacity-100', 'scale-100');
      }
    }, 600);

    // Optional: Hide callout after user starts typing or after 8 seconds
    const hideCallout = () => {
      if (callout) callout.classList.add('opacity-0', 'scale-95');
    };

    saleInput.addEventListener('input', hideCallout, { once: true });
    setTimeout(hideCallout, 8000);
  }
});
Summary – what this gives you

Page loads → cursor is already blinking in the Sale Price field

Soft green pulsing ring appears around the input for ~4 seconds (one gentle breathe cycle)

Callout fades away automatically after user types the first character or after 8 seconds
